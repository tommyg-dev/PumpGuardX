# Security Policy

## Supported Versions

The following versions of PumpGuardX are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of our analysis pipeline seriously. If you discover a vulnerability, please report it following these steps:

1. **Do not disclose publicly.**
2. Send a detailed report to `security@pumpguardx.ai` (simulated).
3. Include a Proof of Concept (PoC) if possible.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

### Scope
- Analysis Engine Logic
- API Gateway Vulnerabilities
- Data Ingestion Anomalies

*Thank you for helping us protect the DeFi community.*
